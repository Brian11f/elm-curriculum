# Introduction to Elm, Lesson 1
## HTML in Elm

### Goals

Students will be able to:
* Model HTML webpages in Elm code
* Manipulate HTML output with Elm data structures

### Syllabus

1. Rendering a Page
    1. Functions
    2. The Virtual DOM
    3. Comments
    4. Exercise 1
2. Basic Data Structures
    1. Strings
    2. Records
    3. Tuples
    4. Lists
    5. Exercise 2
3. Iteration
    1. Booleans
    2. List.filter
    3. List.map
    4. Exercise 3
